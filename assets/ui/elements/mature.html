06 · Highlight
Solid lilac fill, bold serif. The locked rule.
CORRECT · phrases that carry the claim

A reader can skim a page in seconds. Highlight only the load-bearing phrases — the ones that, if removed, would collapse the sentence's meaning. Everything else stays unmarked.

RULES
  1. 1–6 words per highlight. Never full sentences.
  2. One color only — pale lilac. No mixing.
  3. Always Source Serif 4 700, ~1.06× body size.
  4. Chestnut is structural, never a fill.
  5. Ivory paper, never pure white.
