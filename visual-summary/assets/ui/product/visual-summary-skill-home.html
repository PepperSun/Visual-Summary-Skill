Style mode

Output contract

The final delivery stays visual, compact, and structurally explicit.

  • Final format: styled HTML page, not plain text.
  • Voice: compact third-person objective summary, including first-person sources.
  • Section count: short sources use 3 sections; longer sources stay within 3 to 5.
  • Structure primitive: causal or inferential trunks use logic chains; parallel points, cases, and dimensions use tables.
