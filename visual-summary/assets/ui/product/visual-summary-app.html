Default parser scope: text and formulas only. Do not imply semantic understanding of charts, diagrams, or image-first content unless the backend actually supports it.
Backend summary contract
  • Final delivery renders as a styled HTML summary page.
  • Use compact third-person objective voice, including first-person sources.
  • Short sources render in 3 sections; longer sources stay within 3 to 5.
  • Causal or inferential trunks use logic chains; parallel points, cases, and dimensions use tables.
