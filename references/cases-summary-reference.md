## How to use these cases

Use cases to calibrate summary decisions:

- Case 01: explanatory or historical argumentation with staged evolution and cross-sectional mechanisms.
- Case 02: competitive or industry argumentation where the main structure is a flywheel or acceleration chain.
- Case 03: rebuttal persuasion where the first section names the dispute and the main body is a rebuttal table.
- Case 04: fictional narrative where the output should preserve story movement and perceptual abnormality without forcing social interpretation.
- Case 05: first-person essay or memoir-like source summarized in third-person objective voice. If a first-person essay is organized around a debatable claim, classify it as argumentation even when it opens with scenes or anecdotes.
- Case 06: speech-like policy argumentation where the first section states the author's whole-content judgment, and historical evidence supports that judgment later.

Do not copy section titles mechanically. Titles should follow the current source's movement. When in doubt, calibrate section titles here, but do not copy them mechanically.

## Case 01

Source file: `assets/cases/originals/case-01-great-depression-trial.pdf`

### 一、核心结论

大萧条（1929–1941）并非市场自发崩溃，而是多轮政策干预叠加，持续压制投资与市场调整，导致经济长期无法恢复。

### 二、阶段演化（1929–1941）

| 阶段 | 时间 | 强度 | 核心机制（因果链） | 结果 |
| --- | --- | --- | --- | --- |
| 崩盘触发 | 1929–1930 | ★★★ | 货币收缩 → 信用紧缩 → 需求下降 | 经济快速下滑，股市崩盘 |
| 全球断裂 | 1930–1933 | ★★★★ | 关税上升 → 贸易战 → 全球需求崩溃 | 失业达25%，银行危机 |
| 政策延长 | 1933–1937 | ★★★★ | 控制价格/产量 + 加税 → 成本上升 → 投资受抑 | 经济停滞，恢复中断 |
| 二次衰退 | 1937–1941 | ★★★★★ | 劳资冲突 + 政策不确定性 → 资本退出 | 1937–38再次衰退，复苏失败 |

### 三、核心机制（横向总结）

| 机制 | 作用路径 |
| --- | --- |
| 投资被压制 | 税收↑ + 监管↑ + 不确定性↑ → 资本不进入 |
| 价格信号失真 | 人为控制价格/产量 → 市场无法自我调整 |
| 全球循环中断 | 贸易战 → 出口与需求同步下降 |
| 政策不稳定 | 频繁变化 → 企业延迟决策 |

### 四、关键转折与整体感知

| 维度 | 内容 |
| --- | --- |
| 起点 | 1929：股市崩盘，危机显性化 |
| 关键转折 | 1930（贸易收缩）、1933（全面干预）、1937（二次衰退） |
| 终点 | 1941前后：外部需求与政策环境变化带来恢复 |
| 整体路径 | 触发 → 加速 → 延长 → 再次下滑 |
| 总时长 | ≈12年（显著长于常规周期） |

### 五、补充说明（解释偏差）

- 危机的根本原因可追溯到1920年代的货币扩张与随后收缩。
- 但大萧条作为“历史事件”，通常从1929年开始计时。

## Case 02

Source file: `assets/cases/originals/case-02-future-us-china-biotech-competition.pdf`

### 一、核心结论

中国正在从“生物医药制造者”转向“创新驱动者”，并通过制度与资本形成加速系统，这一变化已进入拐点，正在削弱美国的领先地位。

### 二、演化路径（阶段 + 时间感 + 强度）

| 阶段 | 时间感 | 强度 | 核心变化 |
| --- | --- | --- | --- |
| 制造跟随 | 过去20年 | ★★ | 仿制药 + 低成本生产 |
| 创新转折 | 近5年 | ★★★★ | 开始产出创新药 |
| 创新加速 | 当前 | ★★★★★ | 全球药企采购中国创新药 |
| 潜在领先 | 未来10年 | ★★★★★ | 可能成为主要创新中心 |

关键节点：

- 授权占比：5% → 42%（2022→2025）
- 临床试验：中国已超过美国

### 三、核心机制（单一主链）

> 政策 → 成本/速度 → 资本 → 创新 → 再投资

展开：

- 审批加快 → 研发更快
- 成本更低 → 项目更多
- 更早验证 → 更容易融资
- 资本进入 → 创新规模扩大

→ 形成自我强化的创新循环（flywheel）

### 四、系统优势如何转化为领先（关键升级）

把“优势”和“结果”合并成一条链：

> 制度优化 + 资本投入 + 产业协同
>
> → 创新效率提升
>
> → 创新产出增加（IP/临床/授权）
>
> → 全球资本流入中国
>
> → 创新能力进一步放大

关键在于这些优势可以持续放大。

### 五、最终结果（竞争格局变化）

- 创新源头开始向中国转移
- 美国的“研发→资本→创新”体系被削弱
- 生物医药的利润与能力可能外流

趋势判断：

> 若持续，美国可能在数年内失去主导地位。

## Case 03

Source file: `assets/cases/originals/case-03-why-nations-fail.pdf`

### 一、核心争议

Sachs 批评《Why Nations Fail》过度强调制度，忽视地理、资源、威权改革、技术扩散等因素。作者回应说，这些批评多数是误读、概念混淆或证据不足。

### 二、逐条反驳表

| Sachs 的批评 | 作者如何反驳 | 关键证据/例子 | 反驳后留下的判断 |
| --- | --- | --- | --- |
| 威权政府也能推动改革 | 这正是书中已讨论过的情况 | 掠夺性制度下也可能增长 | 威权增长存在，但通常不稳定 |
| 作者混淆创新和技术扩散 | 作者说两者在书中被明确区分 | 中国可通过引进技术增长；但持续创新需要包容性制度 | 扩散≠创新 |
| 地理会影响国家能力和技术扩散 | 作者认为 Sachs 没有给出足够证据 | 回归研究中控制制度后，地理解释力下降 | 地理影响局部条件，但不是主因 |
| 资源决定增长 | 作者用资源国家对比反驳 | 博茨瓦纳有钻石但制度稳定；安哥拉、塞拉利昂、尼日利亚也有资源却表现差 | 资源结果取决于制度 |
| Nogales 是地理案例 | 作者说 Nogales 的关键是政治边界与制度差异 | 同一地区被边界分开后发展不同 | 相近地理下，制度差异更能解释结果 |

### 三、作者反驳方式

作者主要用了三种方法：

| 方法 | 作用 |
| --- | --- |
| 指出误读 | 说明 Sachs 批评的内容其实书中已讨论 |
| 区分概念 | 把“短期增长/长期增长”“扩散/创新”拆开 |
| 举反例 | 用 Botswana、Nogales、中国等案例削弱地理/资源解释 |

### 四、反驳过程中重申的核心观点

其他因素并非无关，但制度决定它们如何发挥作用：

> 资源、地理、技术扩散、威权动员都可能影响发展，但它们是否转化为长期繁荣，取决于制度。

## Case 04

Source file: `assets/cases/originals/case-04-elephant-vanishes.pdf`

### 一、故事梗概

一个在东京近郊生活的上班族，长期关注镇上被安置的一头老象。某天，大象与饲养员在没有任何破坏痕迹的情况下同时消失。主角在此前曾看到两者“比例发生变化”，但无法向他人解释，事件最终被社会遗忘，而这种异常只留在他的感知中。

### 二、故事结构线

| 结构点 | 内容 |
| --- | --- |
| 主角 | 普通上班族，持续关注大象 |
| 初始状态 | 生活有序，世界稳定可理解 |
| 触发事件 | 大象与饲养员突然消失 |
| 推进事件 | 媒体、警方、市政府尝试给出“逃走/被盗”的解释，但均无法自洽 |
| 转折点 | 主角回忆：消失前一晚，大象变小、饲养员变大，比例发生异常 |
| 结尾状态 | 社会逐渐遗忘事件，主角却持续感到世界某种平衡被破坏 |

### 三、情节推动因素

| 驱动因素 | 表现 |
| --- | --- |
| 异常事件 | 大象与饲养员在无任何痕迹下消失 |
| 物理不一致 | 锁链、围栏完好，没有逃离路径 |
| 感知异常 | 主角观察到两者体量比例发生变化 |
| 认知断裂 | 主角无法把这一经历清晰表达给他人，交流失败 |
| 社会消解 | 集体选择接受不完整解释并逐渐遗忘 |

## Case 05

Source file: `assets/cases/originals/case-05-he-caitou-essay.pdf`

### 一、故事梗概

作者回忆自己多年观察到一个现象：很多人搬到北京后逐渐出现花粉过敏，而自己曾长期没有这个问题。直到最近，他也开始出现类似症状，并通过观察发现症状与“开窗通风、接触花粉”密切相关。最终，他把这种变化理解为一种长期环境与生活方式逐渐作用的结果。

### 二、故事结构线

| 结构点 | 内容 |
| --- | --- |
| 主角 | 作者本人 |
| 初始状态 | 长期没有花粉过敏，甚至在云南高花粉环境中也无反应 |
| 触发事件 | 自己开始出现连续打喷嚏、鼻塞等过敏症状 |
| 推进 | 通过对比不同天气、开窗与否等生活细节，逐步发现规律 |
| 转折点 | 确认症状与空气交换、花粉接触直接相关（开窗→发作，封闭→缓解） |
| 结尾状态 | 接受自己已“学会过敏”，并把这一变化归因于长期生活环境与身体适应的改变 |

### 三、情节驱动因素

| 情节驱动因素 | 推动了什么 |
| --- | --- |
| 早期观察（他人过敏现象） | 建立一个长期疑问：为什么有人会“变得过敏” |
| 自身症状出现 | 触发故事进入个人层面 |
| 规律发现（开窗 vs 不开窗） | 推动从“偶发症状”转向“可解释现象” |
| 环境对比（北京 vs 云南） | 强化这是长期环境变化 |
| 过往经验回忆（朋友、医疗手段） | 让现象变得可被接受 |
| 生活方式反思（干净/现代生活） | 推动最终理解的形成 |

## Case 06

Source file: `assets/cases/originals/case-06-waller-speech-modernizing-reserve-bank-operations.pdf`

### 一、核心结论（体现作者偏好）

Waller 主张美联储应走向更深度的运营集中化（Model 2），将当前“统一管理但分散执行”（Model 1）视为过渡方案；其中真正关键的变化不在物理整合，而在于从12家储备银行共识决策，转向系统级统一决策权。

### 二、主结构（压缩后的论点 + 链式论据）

| 论点 | 论据（链式 / 关键支撑） | 推导结果 |
| --- | --- | --- |
| 当前分散结构是历史产物，而非最优设计 | 1913分散设计 → 1935货币政策集中 → 1981成本回收要求 → 1990s IT集中（FRAS）→ 9/11暴露系统脆弱 → Check 21推动数字化 → 2021支付体系集中 | 过去100年一直在向集中化演进 |
| 分散运营已带来成本与效率问题 | IT、HR、采购等功能在12家银行重复建设，且成本显著高于市场水平 | 存在明显规模损失与冗余 |
| 技术与风险环境要求更高集中度 | AI、网络安全、技术周期加快，需要统一架构与快速决策 | 分散 + 慢决策模式不再适用 |
| 职能本质已分化为“本地 vs 系统” | 地区经济研究、监管需本地；IT、支付、财务等为平台型、规模型工作 | 应按性质重新组织，而非按地区划分 |
| 当前治理模式成为瓶颈 | 12家银行共识机制无法支持系统级运营决策 | 必须转向集中决策权 |
| 深度集中化带来效率，但伴随代价 | 集中提高效率与安全，但会带来岗位与组织调整 | Model 2 更优，但需要承担组织成本 |

### 三、嵌套结构：两种运营模式（完整对比）

| 维度 | 模式一：集中管理（过渡） | 模式二：集中化运营（目标） |
| --- | --- | --- |
| 核心逻辑 | 统一领导，分散执行 | 统一领导 + 部分职能集中 |
| 物理布局 | 12家储备银行结构不变 | 非本地职能集中到少数运营中心 |
| 成本效率 | 有改善 | 最大化规模效应 |
| 风险控制 | 改善 | 显著改善（尤其IT/安全） |
| 人员影响 | 基本无变化 | 部分地区岗位减少 |
| 决策模式 | 仍依赖共识 | 系统级负责人拥有实权 |
| 定位 | 过渡方案 | 长期方向 |

### 四、作者的边界声明（预防反驳）

| 作者明确否定的方向 | 含义 |
| --- | --- |
| 不减少储备银行数量 | 保留现有组织框架 |
| 不改变地区划分 | 保持区域代表性 |
| 不把权力集中到华盛顿 | 权力调整针对运营优化，不针对政治集权 |
